## 🪐 Core Features & Cognitive Nodes

### 🧠 1. Interactive Neural Playground (Neural Core)
* **Description**: Directly tap into Nexora's Gemini Pro co-processors for high-throughput text, code compilation, image layout generation, and copywriting metrics.
* **Fallback Systems**: Outfitted with robust fallback co-processors that route requests to an offline sandboxed cyber-simulation if API limits or errors occur, preventing any break in interface continuity.

### 📊 2. Enterprise Growth Suite (Ad Campaigns)
* **Description**: Settle placement budgets, sponsor specific blogs or feed rails, and track campaign performance with fully animated, glowing SVG CTR graphs showing real-time interaction metrics.
* **Dynamic Modals**: Fast campaign deployment tools with real-time impression, click, and click-through-rate simulation vectors.

### 👑 3. Creator Synthesis Network
* **Description**: Seed following bases and simulate revenue on a beautifully designed income dashboard. Enables creators, writers, and developer nodes to plan monetization through interactive sliding parameters for traffic conversion and premium subscribers.

### 🛍️ 4. Secure Digital Storefront (Marketplace)
* **Description**: Instantly acquire developer shell scripts, high-CTR themes, optimal Express routing configs, coding courses, and digital assets. Features integrated search, category filtering, rating logs, and responsive acquisition buttons.

### 🌐 5. Distributed Information Ledger (Trending Hub)
* **Description**: Review real-time community micro-blogs, design manifestos, and quantum logs. Creators can immediately write and publish new posts into the global stream with custom handles and category flags.

### 🌌 6. Premium Orbit Memberships
* **Description**: Unlock scalable resource registers with structured pricing lists. Switch between annual and monthly payment cycles with automated discounts and clear perk indicators.

## 🛠️ Technical Stack & Command Pipeline

Nexora is engineered on a resilient full-stack architecture combining a robust backend with rapid-rendering client screens.

### 📦 Technology Stack
* **Frontend**: React 19, Vite 6, Tailwind CSS 4.0, motion/react, Lucide icons.
* **Backend**: Node.js, Express, TypeScript, Google GenAI SDK (`@google/genai`).
* **Bundler & Dev Engine**: TSX, Esbuild (compiling to self-contained CJS bundle in production).

## 🚀 Commands & Run Guides

### 1. Setup Dependencies
```bash
npm install
```

### 2. Launch Local Development
Starts both the Express API and the Vite Hot Module Replacement (HMR) server side-by-side using `tsx`:
```bash
npm run dev
```

### 3. Production Build
Bundles the frontend using Vite static build and compiles `server.ts` into a fast, compiled CommonJS bundle at `dist/server.cjs` via Esbuild:
```bash
npm run build
```

### 4. Deploy / Production Start
Launches the standalone bundled file server and API:
```bash
npm run start
```
